### Задание 1 (`task1.cpp`)

**Описание:**  
В данном задании реализованы последовательные версии трёх простых алгоритмов сортировки: пузырьком, выбором и вставками. Программа демонстрирует работу каждого алгоритма на одном и том же исходном массиве целых чисел и выводит результаты в консоль.  

**Цель задания:**  
- Освоить реализацию базовых алгоритмов сортировки на языке C++.  
- Научиться работать с динамическими массивами (vector).  
- Измерить и визуализировать результаты работы алгоритмов.  

**Функционал программы:**  
- Создание исходного массива целых чисел.  
- Последовательная сортировка массива пузырьком с выводом результата.  
- Последовательная сортировка массива выбором с выводом результата.  
- Последовательная сортировка массива вставками с выводом результата.  
- Функция `printArray` для вывода содержимого массива.  
- Сброс массива к исходным значениям перед применением следующего алгоритма.

### Задание 2 (`task2.cpp`)

**Описание:**  
В данном задании реализованы параллельные версии трёх алгоритмов сортировки (пузырьком, выбором и вставками) с использованием OpenMP на CPU. Программа создаёт массив случайных чисел, выполняет параллельные сортировки и измеряет время выполнения каждой версии, сравнивая производительность.

**Цель задания:**  
- Освоить параллельное программирование с использованием OpenMP.  
- Научиться распараллеливать циклы и эффективно использовать многопоточность на CPU.  
- Сравнить производительность последовательных и параллельных алгоритмов сортировки.  

**Функционал программы:**  
- Генерация массива случайных чисел в заданном диапазоне.  
- Параллельная сортировка пузырьком (Odd-Even Sort) с распределением итераций цикла между потоками.  
- Параллельная сортировка выбором с поиском минимального элемента в каждом потоке.  
- Параллельная сортировка вставками (Shell Sort) с распараллеливанием подмассивов.  
- Сортировка вставками с ограниченным параллелизмом.  
- Функция `isSorted` для проверки корректности сортировки.
- Измерение времени выполнения каждой сортировки с помощью библиотеки `<chrono>`.  
- Вывод информации о времени выполнения и результате сортировки в консоль.

### Задание 3 (`task3.cpp`)

**Описание:**  
В этом задании реализованы последовательные и параллельные версии сортировок (пузырьком, выбором и вставками) на CPU с использованием OpenMP. Программа проверяет корректность работы алгоритмов и измеряет время выполнения для массивов разного размера, позволяя сравнить эффективность последовательной и параллельной реализации.

**Цель задания:**  
- Продемонстрировать различия в производительности последовательных и параллельных алгоритмов сортировки.  
- Научиться правильно распараллеливать циклы и использовать OpenMP для ускорения работы на CPU.  
- Исследовать эффективность параллельных алгоритмов на массивах различных размеров.  

**Функционал программы:**  
- Генерация массива случайных чисел в заданном диапазоне.  
- Последовательная сортировка пузырьком, выбором и вставками.  
- Параллельная сортировка пузырьком (Odd-Even Sort).  
- Параллельная сортировка выбором с критической секцией для глобального минимума.  
- Параллельная сортировка вставками (Shell Sort) с распараллеливанием подмассивов.  
- Сортировка вставкой с ограниченным параллелизмом.  
- Проверка корректности сортировки функцией `isSorted`.  
- Измерение времени выполнения каждой сортировки с использованием `<chrono>`.  
- Вывод результатов выполнения алгоритмов (время и корректность сортировки) в консоль.
